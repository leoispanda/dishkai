# DishKAI

DishKAI is a lightweight bilingual website MVP for a personalized cross-cultural dining assistant.

## Local development

```bash
npm run dev
```

Open http://127.0.0.1:3000.

## Cloudflare Pages

Deploy the `public` directory. Root files and `public/` copies are kept synchronized for Pages deployment.

Reserved future API endpoints:

- `POST /api/analyze-menu`
- `POST /api/save-profile`
- `POST /api/analyze-receipt`
- `POST /api/save-rating`
